「Unity ゲーム プログラミング・バイブル」サンプルプログラム

No.12 ノベルゲーム
===========================================


【同階層ファイルの説明】
・CreatingDocuments.txt
テキストアセットを作る際に必要な文字をピックアップしたデータです。

・ReferenceDocuments.txt
テキストアセットを作る際に参考なったサイトのURLを記載しております。

・Readme.txt
このファイルです。


【テキストアセットの作成について】
CreatingDocumentsはReferenceDocumentsに記載されているサイトを参考に、
ノベルゲームで必要になりそうな文字をピックアップしました。

TextMesh ProのTextAssetを作成する際は、
CreatingDocumentsの内容をコピーし、Custom Character Listに貼り付けして利用します。

URLにも記載されていない文字は、発見し次第CreatingDocumentsの"self additional"以下に
追記しております。未使用文字は、DofuDetector.csのプログラムにより検出が可能です。
